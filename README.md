## 🚀 Featured Project Directions

### 🧠 RAG-Based Conversational AI for Corporate Learning
- Built a **RAG-based conversational AI prototype** for corporate learning and skills transfer
- Developed a document pipeline for **PDF extraction, normalization, chunking, embeddings, and semantic retrieval**
- Used **Supabase + PostgreSQL + pgvector** for vector storage and knowledge updates without retraining
- Added **persistent memory** and **grounding / refusal policies** to keep responses evidence-based

### 📄 AI-Powered CV Screening & Candidate Ranking
- Built an AI-based **CV screening and ranking system** using **3,334 resumes**
- Developed an NLP pipeline with **cleaning, deduplication, TF-IDF, Truncated SVD, and K-Means**
- Compared multiple ML models and selected **Logistic Regression**, achieving **83% recall** on the positive class
- Designed a **two-stage recommendation system** combining classification and cosine similarity ranking

### 🚦 Urban Mobility Analytics with Big Data
- Built a data-driven system for **road traffic and shared-bike demand analysis**
- Implemented a scalable pipeline with **PySpark, MongoDB, Docker, ETL, and feature engineering**
- Performed **EDA**, weather-demand analysis, and distributed processing
- Trained a **Random Forest Regressor** with **R² = 0.68**

### 🎮 Steam Game Recommendation System
- Built a recommendation prototype on **200k+ user–game interaction records**
- Applied **K-Means, SOM, Apriori, Isolation Forest, LOF, and UMAP**
- Improved recommendation interpretability through **behavioral segmentation and unsupervised learning**
